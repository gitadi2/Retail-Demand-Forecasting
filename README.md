## Overview

This project builds an end-to-end retail demand forecasting pipeline using historical Walmart sales data. The goal is to analyze weekly sales trends, capture seasonality patterns, evaluate forecasting models, and generate future demand predictions to support business planning and inventory optimization.



## Tools & Technologies

- Python (Pandas, Statsmodels, Prophet, Matplotlib)  
- R (forecast package, ARIMA, ETS models)  
- Tableau Public (Interactive Dashboard)


## Key Features

- Performed time series analysis to identify trends, seasonality, and stationarity in retail sales data.  
- Built SARIMA and Prophet models to forecast weekly sales demand.  
- Validated forecast performance using ARIMA and ETS models in R.  
- Generated a 52-week forward sales forecast for demand planning.  
- Designed an interactive Tableau dashboard to visualize actual vs forecasted sales, model performance, and seasonal patterns.  


## Business Impact

- Helps retail teams improve demand visibility and forecasting accuracy.  
- Supports inventory planning and supply chain decision-making.  
- Provides a visual analytics dashboard for business stakeholders.
